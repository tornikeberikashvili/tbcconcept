# თიბისი კონცეფციის საიტი

ეს პროექტი არის TBC Concept-ის საპასუხო ვებგვერდი, სადაც წარმოდგენილია სხვადასხვა პროდუქტები, შეთავაზებები და სერვისები. ვებგვერდი შექმნილია მობილურისთვის მოსახერხებელი და მომხმარებლებს უზრუნველჰყოფს უწყვეტი დათვალიერების გამოცდილებას.

## Მახასიათებლები

- საპასუხო დიზაინი, ოპტიმიზირებულია მობილური მოწყობილობებისთვის
- სათაური ლოგოთი, ნავიგაციის ბმულებით და საძიებო ფუნქციით
- ძირითადი შინაარსი მოიცავს სხვადასხვა სექციებს, როგორიცაა პროდუქტის მაჩვენებლები, შეთავაზებები, სტატისტიკა და ჯილდოები
- ინტერაქტიული სლაიდერები პროდუქტებისა და შეთავაზებების ჩვენებისთვის
- ქვედა კოლონტიტული საკონტაქტო ინფორმაცია და სოციალური მედიის ბმულები

## ტექნოლოგიები

- HTML
- CSS
- JavaScript
- Google ფონტები

## ინსტალაცია

1. საცავის კლონირება:
 ``ბაშ
 git კლონი https://github.com/your-username/tbc-concept-website.git
 ```

2. გადადით პროექტის დირექტორიაში:
 ``ბაშ
 cd tbc-concept-website
 ```

3. გახსენით `index.html` თქვენს სასურველ ბრაუზერში:
 ``ბაშ
 გახსენით index.html
 ```

## გამოყენება

- ვებგვერდის სანახავად უბრალოდ გახსენით `index.html` თქვენს ბრაუზერში.
- დარწმუნდით, რომ გაქვთ ინტერნეტ კავშირი პროექტში გამოყენებული Google ფონტების ჩასატვირთად.
- ვებგვერდი სრულად რეაგირებს და მისი ნახვა შესაძლებელია სხვადასხვა მოწყობილობებზე, მათ შორის მობილურ ტელეფონებზე, ტაბლეტებსა და დესკტოპზე.
